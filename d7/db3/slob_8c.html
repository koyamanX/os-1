Free memory area allocated by kmalloc. @detail Free memory area allocated by kmalloc.

Memory area is maintained in freelist in increasing order of memory address. Search insertion point from start. Insertion points is either one of begining, in-between, or ending. If freed block is adjacent of blocks in freelist, merge them.
